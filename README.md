## About This Project

This is a simple C++ program that generates a file called:

iloveyou.txt

And fills it with thousands of lines:

- I LOVE YOU
- I LOVE YOU
- I LOVE YOU
...

Because sometimes, saying it once is not enough.

## How It Works

The program uses basic file handling in C++:

ofstream to create and write files

A loop to repeat the message thousands of times

## Inspiration

Inspired by the chaotic energy of the early internet,
particularly the ILOVEYOU malware from 2000.

This project does not spread, does not overwrite files,
and definitely won’t ruin your computer…

Probably.
